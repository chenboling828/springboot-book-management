# SpringBoot 图书管理系统

## 项目介绍
基于 SpringBoot + MyBatis + MySQL 实现的简易图书管理系统，支持图书信息的增删改查、模糊搜索等核心功能，适合 Java 后端实习生练手。

## 技术栈
- 后端：SpringBoot 3.2 + MyBatis + MySQL 8.0
- 前端：Thymeleaf + HTML + CSS
- 构建工具：Maven
- JDK：17

## 功能列表
- ✅ 图书列表展示
- ✅ 图书模糊搜索（书名/作者）
- ✅ 新增图书
- ✅ 编辑图书信息
- ✅ 删除图书
- ✅ 操作结果提示
